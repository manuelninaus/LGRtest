# Living Game Repository

Das Living Game Repository ist eine studierendengetragene Sammlung psychologischer Spielanalysen. Es entsteht im Kontext einer Lehrveranstaltung, in der psychologische Theorien auf digitale Spiele angewendet werden.

## Ziel

Das Repository soll zeigen, wie Konzepte aus Psychologie, Medienpsychologie, Lernpsychologie und Motivationspsychologie genutzt werden können, um Videospiele systematisch zu analysieren.

## Proof-of-Concept-Phase

In der ersten Phase dient diese Website als Prototyp. Studierende erstellen Analysen zunächst anhand einer Word- oder Markdown-Vorlage. Geeignete Beiträge werden nach Peer-Feedback und redaktioneller Prüfung in dieses Repository aufgenommen.

## Neue Analyse hinzufügen

1. Datei `templates/analysis-template.md` kopieren.
2. Kopie in den Ordner `_analyses/` legen.
3. Datei sinnvoll benennen, z. B. `minecraft-selbstwirksamkeit.md`.
4. Metadaten im Kopfbereich ausfüllen.
5. Textabschnitte ergänzen.
6. Beitrag zur Prüfung einreichen.

## Struktur

- `index.md`: Startseite
- `about.md`: Projektbeschreibung
- `analyses.md`: Übersicht aller Analysen
- `theories.md`: Theorieübersicht
- `contribute.md`: Anleitung für Studierende
- `ethics.md`: Ethik, Rechte und sensible Inhalte
- `_analyses/`: einzelne Spielanalysen
- `templates/`: Vorlagen für neue Beiträge
- `assets/style.css`: einfaches Design
