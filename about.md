# Über das Living Game Repository

Das Living Game Repository entsteht im Kontext einer Lehrveranstaltung zur psychologischen Analyse digitaler Spiele. Studierende wenden psychologische Theorien auf Videospiele an und entwickeln daraus strukturierte Analysebeiträge.

## Zielsetzung

Das Projekt verfolgt vier Ziele:

1. Psychologische Theorien an konkreten Medienbeispielen anwendbar machen.
2. Studierende beim wissenschaftlichen Schreiben und Argumentieren unterstützen.
3. Wiederverwendbare Lehr- und Transfermaterialien aufbauen.
4. Eine erweiterbare Infrastruktur für spätere Forschung, Lehre und Wissenschaftskommunikation schaffen.

## Proof-of-Concept-Phase

In der ersten Phase steht nicht eine perfekte öffentliche Plattform im Vordergrund. Zentral ist die Frage, ob Studierende theoriegeleitete Spielanalysen erstellen können, die fachlich nachvollziehbar, didaktisch nutzbar und repository-tauglich sind.

## Perspektivische Erweiterung

Nach erfolgreicher Erprobung kann das Repository erweitert werden durch:

- redaktionelle Rollen
- Qualitätsstufen
- öffentliche Showcase-Beiträge
- stärkere Such- und Filterfunktionen
- didaktische Bausteine
- Workshops und Transferformate
- Abschlussarbeiten und Forschungsprojekte
- Analysen weiterer Medienformate
