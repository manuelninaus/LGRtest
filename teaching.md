# Lehre und Transfer

Das Living Game Repository kann in unterschiedlichen Lehr- und Transferkontexten genutzt werden.

## Einsatz in der Hochschullehre

- Einführung in psychologische Theorien
- Anwendung von Theorien auf konkrete Medienbeispiele
- wissenschaftliches Schreiben
- Peer-Feedback und Revision
- Medienpsychologie und Game Studies
- Diskussion von Motivation, Emotion, Lernen und Spielerleben

## Einsatz im Learning Lab

Das Repository kann als Materialpool für Seminare, Workshops, studentische Projekte und Wissenschaftskommunikation dienen.

## Didaktische Bausteine

Aus einzelnen Analysen können kurze Lehrbausteine entstehen:

- Diskussionsfragen
- Mini-Analysen
- Vergleichsaufgaben
- Reflexionsimpulse
- Transferfragen
- Workshop-Materialien

## Perspektiven

Langfristig kann das Repository Grundlage sein für:

- Abschlussarbeiten
- Forschungsprojekte
- Kooperationen mit Schulen
- Austausch mit Kulturinstitutionen
- Wissenschaftskommunikation
- Game-Based-Learning-Projekte
